# Code of Conduct

Be respectful, be constructive, and assume good faith. Harassment, hate
speech, and spam are not tolerated. Always attribute others' work. Maintainers
may edit or remove content that violates these norms or applicable licenses.

Report issues by opening a GitHub issue or contacting the maintainer.
