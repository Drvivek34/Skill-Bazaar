# Writing & Content

Skills for drafting, editing, copywriting, and long-form content.

> Part of **[Skill Bazaar](../README.md)** · Browse all bazaars at the **[Mega AI Bazaar](https://drvivek34.github.io/Mega-AI-Bazaar/)**.

## How entries are stored
Each skill lives in **its own sub-folder** inside this category, named in `kebab-case`:

```
writing-content/
└── <entry-name>/
    ├── README.md     # what it is, source, author, license, link
    └── ...           # the actual files (full copy)
```

## Add to this category
